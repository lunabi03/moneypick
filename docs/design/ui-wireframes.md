# UI 와이어프레임 가이드

텍스트 기반으로 모바일/데스크톱 핵심 화면을 설명합니다. 실제 제작 시 Figma에서 프레임을 구성하고 아래 구조를 참고해 주세요.

## 2. 카테고리 상세 (예: 정부·공공 지원금)

### 헤더 영역

- 상단 탭: `전체 / 청년 / 신혼부부 / 소상공인 / 농어민 / 기타`
- 필터 버튼: `지역`, `소득`, `가구 형태`, `마감일`, `난이도`
- 정렬: 기본값 `추천 순`, 옵션 `마감 임박`, `지원 금액`, `난이도`

### 카드 구성

```
┌──────────────────────────────────────────────────┐
│ [배지: 검증됨]    [마감 D-7]                      │
│ 청년 월세 특별지원                               │
│ - 지원 대상: 19~34세 무주택자, 월세 70만원 이하  │
│ - 지원 금액: 최대 30만원/3개월                  │
│ - 예상 승인율: 78%                               │
│ [Progress bar: 3단계 신청 흐름]                  │
│ CTA: 상세 보기 ┆ 자격 진단                       │
└──────────────────────────────────────────────────┘
```

- 상세 페이지: 신청 절차, 필요 서류, FAQ, 이용 후기, 관련 혜택 추천
- 사이드 패널(데스크톱): 요약 박스(지원 금액, 일정, 제출 서류)

## 3. 검색 결과

### 필터 패널 (데스크톱 좌측 고정 / 모바일 모달)

- 카테고리 체크박스
- 조건 슬라이더: 예상 수익, 난이도, 시간 투자
- 토글: `검증된 정보만`, `신규`, `마감 임박`
- 적용 버튼(Primary), 초기화 버튼(Ghost)

### 결과 리스트

- 컴팩트 카드 (키 비주얼 + 타이틀 + 한 줄 설명 + CTA)
- 키워드 하이라이트 색상: `color.primary`
- 결과 없음 UI: 일러스트 + 대체 추천 CTA

## 4. 저장함 & 알림

### 저장함

```
[상단 탭] 전체 │ 지원금 │ 부업 │ 투자 │ 할인
[필터] 정렬: 추가일/마감 임박
[카드 목록] (간결형)
 ┌───────────────────────────────┐
 │ 정부 청년창업 지원금         │
 │ 즐겨찾기한 날짜, 마감 D-10   │
 │ [CTA: 다시 보기]             │
 └───────────────────────────────┘
```

- 최근 활동 로그, “리마인더 설정” 토글

### 알림함

- 날짜별 그룹핑
- 아이콘(마감, 신규, 추천)
- 읽음/안읽음 상태 시각화
- CTA: 바로가기 / 숨김

## 5. 온보딩 & 자격 진단

### 온보딩 스텝 (모바일 풀스크린)

1. 관심사 선택 (멀티 셀렉트 카드)
2. 기본 정보 입력 (지역, 연령대, 직업)
3. 알림 허용 안내 + 혜택 미리보기

각 스텝 하단: 프로그레스 인디케이터(3단계), 다음 버튼(Primary)

### 1분 자격 진단 Bottom Sheet

```
┌─ 핸들바 ─┐
│ 나에게 맞는 지원금 찾기                         ✕ │
│ Step 1/3                                       │
│ Q. 현재 거주 형태를 선택해 주세요.             │
│ ● 월세  │ ● 전세  │ ● 자가  │ ● 기숙사         │
│ [다음 (Primary)]                               │
└───────────────────────────────────────────────┘
```

- 마지막 스텝 결과: 매칭된 혜택 카드 + 바로 신청 CTA

## 6. 빈 상태 & 에러

- **Empty State**: Illust + “아직 저장된 정보가 없어요” + 추천 액션 버튼
- **Error State**: 회색 배경, 아이콘, 재시도 버튼, 고객센터 링크

---

## 7. 반응형 브레이크포인트 요약

| Breakpoint | Width | 레이아웃 특징 |
| --- | --- | --- |
| Mobile | < 768px | 단일 열, 하단 내비, 플로팅 CTA |
| Tablet | 768~1024px | 2열 그리드, 좌측 고정 필터 |
| Desktop | > 1024px | 3열 그리드, 사이드 정보 패널 |

## 8. Figma 작업 권장 구조

- 페이지 구조: `01_Foundations` → `02_Components` → `03_Pages` → `04_Prototypes`
- Auto Layout을 사용해 카드/리스트를 재사용 가능하게 구성
- Variants: 버튼(3 타입 × 4 상태), 배지(상태별), 입력 필드(기본/포커스/에러)
- Design Tokens는 Styles로 등록 후 코드 연동을 위해 JSON Export 고려
